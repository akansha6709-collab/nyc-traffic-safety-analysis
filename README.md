# NYC Traffic Safety Analysis

## Project Overview
This project analyzes 238,422 NYC motor vehicle collision records to identify accident patterns and key risk factors.

The analysis was conducted using Microsoft Excel with Pivot Tables, conditional formatting, and data visualization techniques.

## Dataset
NYC Motor Vehicle Collisions Dataset

Total Records: 238,422

## Key Questions Explored
- When do most collisions occur during the year?
- What are the peak accident hours?
- What are the major contributing factors to collisions?
- Which factors lead to the most dangerous accidents?

## Key Insights
- Collision activity peaks between **3 PM – 5 PM** (rush hour).
- **March** recorded the highest number of collisions.
- **Driver Inattention/Distraction** is the most common cause of accidents.
- **Failure to Yield Right-of-Way** has the highest percentage of dangerous collisions.
